# KKBOX Song Page (Low-Fidelity Clone)

This is a small static replica of the KKBOX song page layout created with plain HTML, CSS and JavaScript (no frameworks).

How to use:

1. Open `index.html` in a browser (double-click or serve with a static server).

Optional: serve via Python's simple HTTP server for correct file loading:

```bash
python -m http.server 8000
# then open http://localhost:8000/kkbox-clone/
```
